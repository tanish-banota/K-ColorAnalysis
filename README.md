# K-ColorAnalysis
Introduction:

An app that looks at the facial features, skin tone, eye color and hair of a person, it determines your personal color palette and secondary palettes

After determining the color palette, gives a visual of the face and top of the torso

Gives recommendations for jewelry and clothing styles that work with the color palette

Finds stores that would have clothes that match the person's palette

Features:

Having a user profile/Login page

When intially creating profile, saves these 3 attributes
  - Username
  - Password
  - Body type (we give different photos of body types for user to select one)

Profile also saves
  - Color palette/Secondary palettes
  - Clothing style recommendations
  - Jewelry recommendations

Being able to scroll through clothing/jewelry recommendations

Computer Vision to get color values from the face/determine shape of the face
